## PART A: Toxic Link Risk Assessment

### Patterns to Monitor & Avoid

| Risk Type | Description | Prevention |
|-----------|-------------|------------|
| **PBN Links** | Private Blog Networks — cheap links from low-quality sites | Never buy links from Pakistani link farms |
| **Directory Spam** | Mass submission to low-DA directories | Only submit to curated, niche directories |
| **Comment Spam** | Footer/comment links from irrelevant blogs | Use Google Search Console to disavow |
| **Exact-Match Anchor Overload** | >30% exact-match anchors trigger Penguin | Diversify anchors: brand, naked URL, partial match |
| **Single-Niche Blog Networks** | Interlinked blogs all in same niche | Verify host diversity when acquiring links |
| **Press Release Overload** | Mass distribution without editorial value | Limit to 1–2 high-quality PR distributors |
| **Reciprocal Link Farms** | "I link to you, you link to me" swaps | Avoid reciprocal schemes; natural mentions only |
| **Subpar Guest Posts** | Posting on DA<20 sites with no real audience | Target DA>30, real readership, relevant niche |

### Early Warning Signals to Track
- Sudden spike in backlinks from low-DA domains (>50 new links in 1 week from <DA10)
- Anchor text ratio: >40% exact-match keyword anchors
- Links from adult, gambling, pharma, or completely off-topic sites
- Multiple links from same C-class IP range

**Tools to Monitor:** Google Search Console (Links report) · Ahrefs · Moz · SEMrush · Disavow Tool
